Change log

A list of updates made to the business.gov.au style guide.

On this page
  • February 2023
  • January 2023
  • December 2022

February 2023

Change Details Relevant designs Date

Update stepped navigation breakpoints.

Added an additional breakpoint to the stepped navigation component between 768px and 992px. Within this breakpoint the step name font size drops to 13px and the right-margin between steps is reduced from 16px to 4px.
  • Stepped navigation

14/02/2023

Update scrollable modal width.

Updated the maximum width of scrollable modals that contain a lage amount of content.

The maximum width of these modals is 900px (instead of 640px).
  • Modals

10/02/2023

January 2023

Change Details Relevant designs Date

Update stepped navigation step states.

Updated the 'active' and 'completed' step states in the stepped navigation.

The active step is styled with a bold font-weight and no underline. The completed steps are styled with a semibold font-weight and an underline.
  • Stepped navigation

31/01/2023

Update guided search single select padding.

Updated the padding on the single select in the guided search. Top and bottom padding is now 12px.
  • Guided search single select questions

31/01/2023

Update toggle focus state.

Updated the focus state for toggles in the guided search.

The new focus state is a 2px solid blue border (#2E6CB9). This change was made to retain consistency with other focus states.
  • Guided search toggle questions

31/01/2023

Update list padding.

Padding for ordered and unordered lists has been updated on screen sizes less than 576px wide. The left list padding is now 32px (previously 24px).
  • Typography

31/01/2023

Update the tile filters search input and button size.

The button size for search bars in tile filters has been updated to 56px x 56px. The height of the search input field is also 56px. This change was made to retain consistency with other inputs across the site.
  • Tile filters

30/01/2023

Change to the pathway accordion chevron padding

Changed the padding on either side of the pathway accordion chevrons at mobile size. The padding is now 16px both left and right of chevron.
  • Pathway accordion

30/01/2023

Update chat button 'closed' state trigger

Changed the trigger state for the closed chat button. The pop-up is now triggered on 'click' rather than on 'hover'. This is due to the inclusion of a link in the pop-up message. In the hover state it is hard (or impossible) for the user to mouse over the link.
  • Chat button

20/01/2023

Add additional breakpoint to grant rounds accordion

Added an extra breakpoint to the grant rounds accordion (below 768px). Below 768px the elements in the rows wrap differently with the round number and status tag displaying above the page link.
  • Grant rounds accordion

19/01/2023

Create rules for button group spacing

Created rules for spacing between buttons and links when grouped together.

Groups of buttons and links have a row-gap of 24px and a column-gap of 24px.

Below screen widths of 576px the font-size of links and buttons in button groups is 14px with a line-height of 28px.

Below screen widths of 350px buttons and links are 100% wide.
  • Buttons and links
  • Guided search
  • Subscribe component
  • Modal dialog
  • Business.gov.au site footer
  • In page feedback form

10/01/2023

Update focus state on pagination links

Changed the background colour and border on the pagination link focus state. Pagination links focus state is now a white background with 2px blue border (#2E6CB9).
  • Pagination

09/01/2023

December 2022

Change Details Relevant designs Date

Update padding on checkboxes in the tile filters.

Updated the left padding on secondary checkboxes in the tile filters to match the padding on default form checkboxes.

Secondary checkboxes now have 8px padding to the left.
  • Tile filters

15/12/2022

Update feature boxes in the lightweight checklist.

Updated padding in the feature box when it appears in the lightweight checklist.

Feature box padding (in this context) is 16px left and 24px right, top and bottom. Padding between the icon and the text is 16px.
  • Lightweight checklist accordion

15/12/2022

Update in-text link hover & focus states.

Changed the hover and focus state background colour on in-text links to a transparent colour. This is to ensure the same colour can be used in both white and coloured background sections. The new colour is rgba(46, 108, 185, 0.15).

Changed the font colour in the hover and focus states to #113869.
  • In-text link

14/12/2022

Update sticky header padding.

Changed the top and bottom padding on the finder tool 'sticky' headers to avoid the header flickering when the user scrolls past the sticking point.

The sticky header now has 32px top and bottom on large screens. On small screens the padding is 16px above and 24px below.
  • Showing number sticky header

13/12/2022

Update home banner image size on smaller screen sizes.

Added additional breakpoints for image height in the home banner.

The new image heights are:

  • Between 768px and 576px - max height 384px
  • Between 576px and 350px - max height 272px
  • Below 350px - max height 176px
  • Home page banner

9/12/2022

Update social media icon size in site footer.

Removed an inconsistency in the design specifications where the social media icons were displaying at 2 different sizes.

The revised icon size is 48px wide x 48px high (including the 2px border).
  • business.gov.au site footer

7/12/2022
