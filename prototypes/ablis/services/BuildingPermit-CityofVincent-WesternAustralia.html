Overview

You will need this permit if you intend to conduct building work.

Building work is broadly defined. You should consult with the relevant Council to determine the exemptions that apply.

There are two types of building permit applications, 'certified' and 'uncertified'.

Certified application - this requires the permit application to be accompanied by a certificate of design compliance issued by a building surveyor. Permit applications for Class 1b and Class 2 to Class 9 buildings must be made as certified applications, whereas this is optional for Class 1a and 10 buildings and incidental structures.

Uncertified application - these applications can only be made for Class 1a and Class 10 buildings and incidental structures. An uncertified application is not required to be accompanied by a certificate of design compliance.

Eligibility

Please contact the council to identify any eligibility requirements.
