Guidelines for use

Purpose:

The tool finalise call to action is used to provide users with the ability to:

  • download a document
  • email a document
  • save their progress within the tool

The component includes a verification process for verifying the users email address.

When to use this component:

  • Used in the Employment Contract Tool.
  • This component could be adapted to saving tool progress or downloading / emailing documents in other contexts.

Version history

26 March, 2024
