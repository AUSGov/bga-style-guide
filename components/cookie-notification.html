Guidelines for use

Purpose:

The cookie notification is displayed at the bottom of the screen when users first visit business.gov.au.

This notification does not stop the user from viewing and using the pages on business.gov.au. The notification remains in place until the users clicks on either "Accept all cookies" or "Customise cookies". Once the user has clicked on one of these links they do not see the cookie notification again.

The "Customise cookies" link leads to a page on business.gov.au about cookies. The page should include the ability to turn off optional cookies. Ensure users can access this page from elsewhere where on the site , e.g. a link in the footer.

When to use this component:

  • Used on all pages of business.gov.au for users that have not seen and actioned the cookie notification before.

When not to use this component:

  • Not used for any purpose on business.gov.au other than conveying information about cookies. If a pop up notification is required for other reasons use the toast notification.
