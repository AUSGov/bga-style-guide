Guidelines for use

Purpose:

Used to provide a link to the business.gov.au online chat service.

The chat button has an open and closed state. Hovering over the closed button or clicking on the closed button on mobile screens displays a pop up message explaining when the chat is open.

When to use this component:

  • Used with all pages of the business.gov.au site.
