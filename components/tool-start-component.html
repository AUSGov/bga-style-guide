Guidelines for use

Purpose: 

The start component is designed to highlight a tool start button while also providing a short description of the tool's purpoose

When to use this component: 

  • Start components can be used on tool landing pages. They should only ever appear on the first page of a tool or process.

Content rules: 

  • Descriptions should focus on what the user can use the tool for
  • Less important information can be displayed under the start button box.
  • When used placed in page (rather than the header) the tool start component should be the first item in the content.
