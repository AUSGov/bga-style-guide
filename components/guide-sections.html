Guidelines for use

Purpose:

Guide sections are used to present a set of information tiles relating to a specific topic. A business.gov.au guide consists of a number of sections outlining information needed for completing a specific user journey such as starting a business.

When to use this component:

  • Used in the business.gov.au guide pattern.
