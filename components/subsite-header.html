Guidelines for use

Purpose:

The subsite header provides branding for BGA subsites and a link to the main homepage of business.gov.au. It can also provide additional subsite links if required.

When to use this component:

  • Used across every page of a busniess.gov.au subsite or tool to ensure a consistent user experience.
