Guidelines for use

Purpose:

Provide customised links to support services in the context of a page

When to use this component:

  • To link to finder tools with appropriate filters pre-selected from relevant pages.
  • To link from a grants page to other grants the user has previously visited

Version history

8 August, 2023

  • Added an additional layout (the two column - right sidebar alternate layout).
