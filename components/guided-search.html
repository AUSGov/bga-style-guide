Guidelines for use

Purpose:

The guided search pattern steps users through a series of questions to help them refine a set of results.

Designed to be used in conjunction with tile filters to provide alternate ways of filtering information. When used in this way the guided search and filters work together to provide real time filtering of a set of results. An example of this design pattern can be seen in the business.gov.au grants and programs finder.

When to use this component:

  • Used in tools or search pages.
