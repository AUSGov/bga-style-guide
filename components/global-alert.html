Guidelines for use

Purpose:

Global alerts warn users about current issues on the site with a message at the top of every page.

Global alerts can be dismissed by the user with a close button in the top right corner. Once the user has dismissed a global alert it should not reappear on other pages of the site.

When to use this component:

  • Typically used for messages about scheduled site maintenance or releases.
