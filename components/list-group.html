Guidelines for use

Purpose:

List groups display information in an easy to scan grid of rows and columns.

When to use this component:

  • For large lists that would be easier to scan when displayed in a grid format.

Version history

8 August, 2023

  • Added an additional layout (the Two column - tool page alternate layout).
