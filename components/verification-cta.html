Guidelines for use

Purpose:

The verification call to action component is used to allow users to verify their email address.

It consists of a two-step process where a user enters their email address and then enters a code that has been sent to that email address. If the users' email address is already known they are asked to send themselves the code without having to enter their email address.

When to use this component:

  • Used in the Employment Contract Tool to allow users to save their progress in the tool for seven days at a time.
