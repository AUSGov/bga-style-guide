Guidelines for use

Purpose

The in page feedback form is used to capture simple feedback from users in the context of a page.

When to use this component:

  • Can be used on all business.gov.au pages.

Version history

8 August, 2023

  • Added an additional layout (the Two column - tool page alternate layout).
