Guidelines for use

Purpose:

The save and return footer is used in business.gov.au tools to provide a lightweight way to save progress in a tool without an account. It allows users to save their progress against a verified email address.

When to use this component:

  • Used in the Employment Contract Tool category pages.
  • This component could be adapted for other tools. When used in a tool it should be placed on every category page to allow users to save their progress at any point while they are working through a process.
