Guidelines for use

Purpose:

An accordion is an expandable section with nested content. This component provides content details when opened by users on a content page. It can help to streamline the user experience by keeping content relevant to the user.

When to use this component:

  • For optional content (e.g. frequently asked questions at the end of a transaction page).
  • For directory type content (e.g. state and territory links).
  • To streamline a transaction process.

Content rules:

  • Select “open all” accordion option if you have 2 or more accordion items.
  • Avoid single accordion items, although they are sometimes used on grant application pages to streamline the application process.
  • Avoid using bold in accordion body content.
