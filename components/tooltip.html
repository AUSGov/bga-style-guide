Guidelines for use

Purpose:

Used to provide additional information about an element when the user moves the mouse pointer over it.

When to use this component:

  • Used within tool pages for external links that open in new tabs. Note tooltips are not used for external links in general business.gov.au information pages.
  • Used with the chat button when the chat feature is unavailable.
  • Used with the print and share utilities component to provide links to different social media sites listed under the 'share' link.

Style rules

Tooltips can be positioned above, below or to the side of their trigger element. By default tooltips should display above the trigger element. The tooltip used in the print and share utilities component displays below the trigger link.

Tooltips can be triggered on hover or click:

  • The chat button and print and share utilities component tooltips have a click trigger.
  • External link tooltips have a hover trigger.
