Guidelines for use

Purpose:

Used in the Country of Origin Labelling (CoOL) tool to help users create country of origin labelling for food products. Depending on user requirements the create label cta will:

  • generate a custom standard mark label, or
  • provide guidance on how to country of origin statement requirements.
