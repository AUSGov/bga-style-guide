Guidelines for use

Purpose:

The ABLIS header provides branding for ABLIS. It can also provide additional links when appropriate:

  • In tool pages the name of the tool is included underneath the logo (either "Business Licence and regulation finder" or "Keyword search"
  • If the user has previously visited business.gov.au in the same session the 'return to BGA' link is displayed. This link is not present if the user has not come from BGA.

When to use this component:

  • Used across every page of ABLIS to ensure a consistent user experience.
