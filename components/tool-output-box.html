Guidelines for use

Purpose:

Displays information about regulations or legal clauses. Tool output boxes can incorporate an "included" tag to indicate that the user has included the clause in their process.

When to use this component:

  • Primarily used in tools to highlight legal information.

Version history

16 February, 2024

  • Updated the component name from clause /regulation box to 'tool output box'.

22 August, 2023

  • Updated the padding and corner radius on tile tags. Corner radius is 4px. Padding is 4px above and below and 8px on the left and right. Padding measurements include border widths when tags have a visible border.

8 August, 2023

  • Added an additional layout (the Two column - tool page alternate layout).
