Guidelines for use

Purpose:

The contract actions is used in tools to provide users with the ability to:

  • return to a previously saved progress
  • extend the expiry date on saved progress
  • delete their saved progress.

When to use this component:

  • Used in the Employment Contract Tool to display saved contracts. Contracts are saved for seven days at a time with the ability to extend the expiry date twice.
