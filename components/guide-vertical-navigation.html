Guidelines for use

Purpose:

Stepped navigation guides users through a process that is completed in a set order. It displays where the user is up to and allows then to navigate back to previous steps.

Users cannot navigate forward to uncompleted steps using this component.

When to use this component:

  • Used in tools or forms that require the user to complete a series of steps.
