Guidelines for use

Purpose: 

The image pathway list component is used to provide pathways to a list of pages on the site, using tiles with a supporting image. It has an optional button below the image pathways. 

When to use this component: 

  • Image pathway components are commonly used on the home, news and customer story index pages. They provide navigation tiles (each with an image) to a list of similar page types (such as news articles or customer stories).  
  • Image pathways may also be used on content pages for medium priority tasks (for example, to list customer stories for a particular grant). 

Content rules: 

  • Display description for each item. 
  • No alt text for pathway images.
