Map controls

Google maps provides a number of possible map controls. On business.gov.au include only two control types:

  • The default zoom controls - plus and minus buttons that allow the user to zoom in and out of the map.
  • A custom 'reset map' button - returns the user to the original map position.

For more information on Google map controls visit the documentation on map controls.

Marker clusters

Google map clusters are used when a large number of pins occur close together. Users can click on a cluster to zoom in and see individual pin locations.

On business.gov.au cluster colours and fonts are customised but sizing and behaviours use Google defaults.

For more information on Google marker clusters visit the documentation on marker clustering.

Location marker

business.gov.au uses a custom marker to show the location of business advisers. Clicking on a marker reveals the search result tile for that adviser.

If maps are used in other situations consider using a custom marker that uses a colour from the business.gov.au colour palette. For more information on custom markers visit the documentation on custom markers.
