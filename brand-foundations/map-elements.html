Map controls

Google maps provides a number of possible map controls. On business.gov.au include only two control types:

  • The default zoom controls - plus and minus buttons that allow the user to zoom in and out of the map.
  • A custom 'reset map' button - returns the user to the original map position.

For more information on Google map controls visit the Google maps platform documentation.

Marker clusters

Google map clusters are used when a large number of pins occur close together. Users can click on a cluster to zoom in and see individual pin locations.

On business.gov.au cluster colours and fonts are customised but sizing and behaviours use Google defaults.

For more information on Google marker clusters visit the Google maps platform documentation.
