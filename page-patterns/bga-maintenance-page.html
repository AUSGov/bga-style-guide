Guidelines for use

Purpose

The maintenance page is used when business.gov.au is down for scheduled maintenance.

It should always include:

  • The red 'error' header including the alert icon.
  • Information on contacting the business.gov.au call centre.

It should not include:

  • The business.gov.au site header. Instead just include the business.gov.au logo.
  • Breadcrumbs
  • The business.gov.au site footer.

Related components
