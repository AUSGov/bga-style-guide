Guidelines for use

Purpose

An adviser page is a single page detailing one adviser per page.

It should always include:

  • Any relevant contact information
  • Brief description about the skills and background of the adviser (try to limit to paragraphs to keep it consistent with adviser s across all areas)
  • A maximum 90 character metadata description for search (this does not appear on the page)
